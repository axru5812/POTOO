TrainingFunctions¶

Module with functions related to training the sklearn ML model.

TrainingFunctions._add_single_noise(value)[source]¶

Adds 10 percent gaussian noise to a number. Utility function for use in add_noise

TrainingFunctions.add_noise(datafile, responsefile, overwrite=True, save_x=None, save_y=None)[source]¶

Adds a gaussian noise component to the cloudy modelruns, effectively doubling the training set. Saves the resulting dfs as pickles

Parameters:
  • datafile (str) – file containing the main dataset pickle
  • responsefile (str) – file containing the y classes
  • overwrite (bool) – whether to overwrite the old files
  • save_x (str) – If overwrite is false, this specifies the savelocation of the data
  • save_y (str) – If overwrite is false, this specifies the savelocation of the responses
TrainingFunctions.compile_training_set(data_dir, save_file_int, save_file_class, clobber=False, linelist='./data/linelabels')[source]¶

Constructs a data-frame containing the line flux values. The resulting DF is then saved as a pickle. The same is done with the class values

Parameters:
  • data_dir (str) – Directory from which to fetch the cloudy savefiles
  • save_file_int (str) – Name of the file to which to save the pickled dataframe
  • save_file_class (str) – Name of the file to which to save the classlabel dataframe
  • linelist (str) – Path to the list containing the linelabels. Defaults to ‘./data/linelabels’
TrainingFunctions.cross_validate(data, response, model=None)[source]¶

Run a k-fold cross validation on the model to assess predictive ability.

TrainingFunctions.load_data(x_file, y_file, standardize=True, standardizer_file='./data/standardizer.pkl')[source]¶

Loads and returns the data. If standardize is True (default) it standardizes the data and persists the standard scaler object for use on the actual dataset

Parameters:
  • x_file (str) – Name of the pickle containing the training data set.
  • y_file (str) – Name of the pickle containing the training classes.
  • standardize (bool) – Whether or not to standardize the data
  • standardizer_file (str) – Name of file in which to persist the data
Returns:

  • x_df (pandas.DataFrame) – training data set
  • y_df (pandas.DataFrame) – training classes

Trains the selected model on the full data set

  • response (pandas.DataFrame) – Classes from Cloudy column density
  • model (sklearn model object, optional) – An instance of an sklearn classifier. If not specified the function defaults to an AdaBoostClassifier
